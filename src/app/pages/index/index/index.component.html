Index

Angular で作ったツールやサンプルを公開しています。

Font Family Tester

font-family に指定するフォントを動的に追加・変更・削除してフォントの適用具合を確認できるツールです。

Multiple Diff

3ファイル以上同時に比較して Diff が確認できます。

Calculator

入力値を計算します。

RegExp

正規表現を試せます。

Text Converter

テキスト変換系のツールです。

Add Line Number

入力されたテキストに行番号を付与します。

Case Converter

入力されたテキストを様々なケースに変換します。

Beautifier

各種コードを整形するツール群です。

JavaScript Beautifier

JavaScript コードを整形します。JSON データも整形できます。

HTML Beautifier

HTML コードを整形します。HTML 中の CSS・JavaScript も整形できます。

CSS Beautifier

CSS コードを整形します。SCSS コードも整形できます。

CSV File To Table

CSV ファイルを HTML テーブル形式で表示するサンプルです。ファイルのドラッグ & ドロップに対応しています。

Touch Friendly DnD

PC でもスマホでも使える DnD を検証しました。

Dynamic Generate Form

JSON で定義したモデル情報に基づいてフォームを自動生成します。
